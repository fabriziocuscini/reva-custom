# Layout Pattern Facades Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Expose all 20 Panda CSS layout patterns as facade components from `@reva/ui`, making it the single import source for consumers.

**Architecture:** Each layout component is a thin facade module (owned props interface + component assignment + displayName) re-exporting from Panda's generated `styled-system/jsx`. A single barrel file (`src/index.ts`) exposes everything from the package root.

**Tech Stack:** TypeScript, Panda CSS (codegen), tsdown (build)

**Spec:** `docs/superpowers/specs/2026-04-05-layout-pattern-facades-design.md`
