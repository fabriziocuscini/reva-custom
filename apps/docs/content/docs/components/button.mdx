---
title: Button
description: A button component with solid, outline, and ghost variants.
---

## Usage

Import the Button component from `@reva/ui`:

```tsx
import { Button } from '@reva/ui'
```

## Variants

The Button supports three variants: `solid` (default), `outline`, and `ghost`.

## Sizes

Three sizes are available: `sm`, `md` (default), and `lg`.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'solid' \| 'outline' \| 'ghost'` | `'solid'` | Visual style |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Button size |
| `className` | `string` | — | Additional CSS classes |
