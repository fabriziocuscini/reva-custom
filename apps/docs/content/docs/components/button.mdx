## Usage

```tsx
import { Button } from '@reva/ui'
```

## Examples

### Variants

The Button supports four visual styles: `solid` (default), `subtle`, `outline`, and `ghost`.

<ComponentPreview code={variantsCode}>
  <ButtonVariants />
</ComponentPreview>

### Colors

Six color palettes are available: `accent` (default), `neutral`, `error`, `success`, `warning`, and `info`.

<ComponentPreview code={colorsCode}>
  <ButtonColors />
</ComponentPreview>

### Sizes

Four sizes are available: `xs`, `sm`, `md` (default), and `lg`.

<ComponentPreview code={sizesCode}>
  <ButtonSizes />
</ComponentPreview>

### Disabled

Use the `disabled` prop to prevent interaction.

<ComponentPreview code={disabledCode}>
  <ButtonDisabled />
</ComponentPreview>

## Props

| Prop        | Type                                                                   | Default    | Description            |
| ----------- | ---------------------------------------------------------------------- | ---------- | ---------------------- |
| `variant`   | `'solid' \| 'subtle' \| 'outline' \| 'ghost'`                          | `'solid'`  | Visual style           |
| `color`     | `'accent' \| 'neutral' \| 'error' \| 'success' \| 'warning' \| 'info'` | `'accent'` | Color palette          |
| `size`      | `'xs' \| 'sm' \| 'md' \| 'lg'`                                         | `'md'`     | Button size            |
| `className` | `string`                                                               | —          | Additional CSS classes |
| `disabled`  | `boolean`                                                              | `false`    | Disable the button     |
